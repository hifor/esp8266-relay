# esp8266-relay
WIFI开关





## 连线

| Relay             | ESP8266       |
| ----------------- | ------------- |
| IN (高低电平控制) | D1引脚  GPIO5 |
| +(电源) VCC       | VIN           |
| -(电源) GND       | GND           |



| Relay                   | lamp(负载)    |
| ----------------------- | ------------- |
| COM 公共端              | L 市电火线 |
| NO 常开端(与公共端断开) | L 连接到负载的火线 |
| NC 常闭端(与公共端连通) |            |
